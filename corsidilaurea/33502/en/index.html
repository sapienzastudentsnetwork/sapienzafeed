25/06/2026 - Attestation for Lazio DiSCo

Please be reminded that the DISCO Lazio call requires that, for the purposes of issuing certificates, the number of ECTS credits (CFU) must be clearly specified for each module.
If requests are not correctly completed, this information will be missing, and it will not be possible to proceed with the issuance of the certificate.
It should also be noted that, to date, only a very small number of students have requested documentation including the specific CFU value for each module.
Finally, please be reminded that the certificate is issued exclusively upon student request, and that students are responsible for carefully consulting the relevant call. 

Each email must include the following information and must be sent exclusively from the Sapienza institutional email address:
 

  • Subject: Request for Attestation for Lazio DiSCo
  • First and last name
  • Student ID number (matricola)
  • Degree programme enrolled in
  • Course modules with all details (course code and ECTS credits for each module)
  • Course title with code
  • Lecturer
  • Exam date
18/06/2026 - Request for a DiSCoLazio scholarship certificate

CERTIFICATE REQUEST FOR COURSES STRUCTURED INTO TWO UNITS
(How to write the email request for a DiSCoLazio scholarship certificate)

To obtain an official certificate confirming the completion of a single module (Unit 1 or Unit 2) of a course structured into two units, students must send an email including all the mandatory information listed below.

  EMAIL SUBJECT to segr.didattica@di.uniroma1.it

Request for module completion certificate – DiSCoLazio scholarship

  MANDATORY INFORMATION TO INCLUDE IN THE EMAIL

The email must include: 

  • Sapienza institutional email address
  • full name, 
  • student ID number (matricola)
  • Degree programme
  • year of study (I, II, III, etc.)

Indicate the exact official course title as listed in the Study Programme (Degree Catalogue) of your Degree Course, in the “Attendance” section, then in “Courses”:
https://www.di.uniroma1.it/it/didattica-informatica

  • Course code (as per the official programme of the Degree Course)
  • Full official course title
  • Module taken (Unit 1 / Unit 2)
  • Name of the instructor with whom the module exam was taken
  • Date of the exam
